# RVC News Wall

This repository hosts the official RVC News Wall portal, published through GitHub Pages.

## Purpose

This repository provides a centralized news dashboard for quick access to major Brazilian live news channels.

The platform serves as a personal news monitoring portal, allowing direct access to live broadcasts from multiple trusted news sources through a simple and responsive interface.

## Files

```text
index.html      → Main news portal page
CNAME           → Custom domain mapping for GitHub Pages
README.md       → Project documentation
favicon.ico     → Website favicon (optional)
assets/*        → Branding resources and channel logos (optional)
.nojekyll       → Disables Jekyll processing for GitHub Pages compatibility
```

## Notes

This repository is intended exclusively for the public RVC News Wall portal and related branding resources.
